# VPN-guide

För att installera VPN:

1. Ladda ner företagets godkända VPN-klient.
2. Logga in med ditt arbetskonto.
3. Slutför tvåfaktorsautentisering.
4. Välj rätt serverprofil.
5. Kontrollera att anslutningen visar status Connected.

Om VPN inte fungerar:
- logga ut och in igen
- kontrollera tvåfaktor
- verifiera att rätt server används
