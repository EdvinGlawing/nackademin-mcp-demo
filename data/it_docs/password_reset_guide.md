# Guide för lösenordsåterställning

Om du inte kan logga in:

1. Kontrollera att du använder rätt användarnamn.
2. Återställ lösenord via lösenordsportalen.
3. Om kontot är låst, vänta några minuter och försök igen.
4. Testa att logga in i ett privat webbfönster.
5. Kontakta helpdesk om problemet fortsätter.

Om tvåfaktorsautentisering är aktiverad måste även den slutföras korrekt.
