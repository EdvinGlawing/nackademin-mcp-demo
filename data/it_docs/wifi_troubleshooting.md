# Wi-Fi felsökning

Om datorn inte hittar Wi-Fi:

1. Kontrollera att trådlöst nätverk är aktiverat.
2. Starta om datorn.
3. Kontrollera att flygplansläge inte är aktiverat.
4. Glöm nätverket och anslut igen.
5. Testa ett annat nätverk för att avgöra om felet gäller datorn eller nätverket.

Om problemet kvarstår ska helpdesk kontaktas.
